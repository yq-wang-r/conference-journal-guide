# 综合领域期刊和会议数据

## AI 领域 (Artificial Intelligence & Machine Learning)

### 顶级期刊
1. **Journal of Machine Learning Research (JMLR)** - 开源期刊，影响因子高
   - 网址: https://www.jmlr.org/
   - 主题: 机器学习、统计学习、算法、应用
   - 难度: 高

2. **Artificial Intelligence (Elsevier)** - 顶级期刊
   - 网址: https://www.sciencedirect.com/journal/artificial-intelligence
   - 影响因子: 9.088
   - 主题: AI基础、知识表示、推理、规划

3. **IEEE Transactions on Pattern Analysis and Machine Intelligence (TPAMI)**
   - 网址: https://ieeexplore.ieee.org/xpl/RecentIssue.jsp?punumber=34
   - 影响因子: 很高
   - 主题: 模式识别、计算机视觉、机器学习

4. **Neural Networks (Elsevier)**
   - 网址: https://www.sciencedirect.com/journal/neural-networks
   - 影响因子: 8.0+
   - 主题: 神经网络、深度学习、学习算法

5. **IEEE Transactions on Neural Networks and Learning Systems**
   - 网址: https://ieeexplore.ieee.org/xpl/RecentIssue.jsp?punumber=5962385
   - 主题: 神经网络、学习系统、自适应系统

6. **Machine Learning (Springer)**
   - 网址: https://link.springer.com/journal/10994
   - 主题: 机器学习理论与应用

7. **Information Fusion (Elsevier)**
   - 网址: https://www.sciencedirect.com/journal/information-fusion
   - 影响因子: 12.975
   - 主题: 数据融合、多源信息处理

8. **Pattern Recognition (Elsevier)**
   - 网址: https://www.sciencedirect.com/journal/pattern-recognition
   - 主题: 模式识别、计算机视觉、图像处理

9. **ACM Transactions on Machine Learning Research (TMLR)**
   - 网址: https://openreview.net/group?id=TMLR
   - 主题: 机器学习研究

10. **Foundations and Trends in Machine Learning**
    - 网址: https://www.nowpublishers.com/mal
    - 影响因子: 25.4
    - 主题: 机器学习综述和基础
